## 🚀 Quick Start

1. Clone this repository
2. Open `index.html` in your browser
3. Start managing your tasks!

## 💡 How to Use

1. Type your task in the input field
2. Press "Add Task" button or hit Enter
3. To edit a task, click the edit (✏️) button
4. To delete a task, click the delete (🗑️) button
